# BioMentor

A simple static website for learning biology fundamentals.

## Preview locally

Use either command below from the project folder:

```bash
npm run preview
```

or

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.

## If preview is not working

- Make sure you run the command inside this repository folder.
- Confirm port `8000` is free, or stop any process already using it.
- If your environment needs network binding, run:

```bash
npm run preview:bind
```
